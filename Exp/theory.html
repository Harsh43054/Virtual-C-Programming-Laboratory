Introduction to C++

C++ is a powerful general-purpose programming language developed by Bjarne Stroustrup. It supports procedural, object-oriented, and generic programming paradigms and is widely used in software development, game development, competitive programming, and system applications.

Structure of a C++ Program

A basic C++ program consists of a header file, namespace declaration, main() function, executable statements, and a return statement. The execution of every C++ program begins from the main() function.

Header Files

Header files contain declarations for functions, classes, and libraries. The <iostream> header provides input and output functionality using cin and cout.

Namespace

The statement using namespace std; allows programmers to access standard library objects and functions without repeatedly using the std:: prefix.

Main Function

The main() function is the entry point of every C++ program. Program execution starts from this function and ends when it returns a value.

Input and Output Statements

C++ uses cin for taking input from the user and cout for displaying output on the screen. These are defined in the iostream library.

Return Statement

The statement return 0; indicates successful completion of the program and returns control to the operating system.

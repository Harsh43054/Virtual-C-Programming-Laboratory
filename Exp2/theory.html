Input, Output and Data Types Explorer

Input and Output in C++

Input and Output operations allow a program to communicate with users. In C++, the cin object is used to accept input from the keyboard, while the cout object is used to display output on the screen.

Data Types

  • int: Stores integer values (e.g., 10, 25, -5)
  • float: Stores decimal numbers (e.g., 3.14, 2.5)
  • double: Stores large decimal values with higher precision
  • char: Stores a single character (e.g., 'A', 'x')
  • bool: Stores logical values (true or false)

Variables

Variables are named memory locations used to store data. Before using a variable, it must be declared with an appropriate data type.

Example:
int age = 20;

Input using cin

The cin statement allows users to enter values during program execution.

Example:
cin >> age;

Output using cout

The cout statement displays text and variable values on the console screen.

Example:
cout << age;

Program Execution Flow

A C++ program starts execution from the main() function. The program accepts input, processes data, displays output, and finally terminates using return 0;.

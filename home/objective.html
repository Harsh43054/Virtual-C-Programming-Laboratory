Computer Science and Engineering

C++ Programming Lab

  1. To understand and apply fundamental programming concepts using C++.
  2. To write, compile, and execute C++ programs using variables, data types, operators, and control structures.
  3. To develop problem-solving skills using functions, arrays, strings, and recursion.
  4. To understand and implement pointers and dynamic memory allocation.
  5. To create and use classes and objects for object-oriented programming.
  6. To understand the principles of object-oriented programming including encapsulation, inheritance, and polymorphism.
  7. To implement file handling techniques for reading from and writing to files.
  8. To use exception handling mechanisms for managing runtime errors effectively.
  9. To develop logical thinking, debugging, and program-testing skills through practical exercises.
  10. To build confidence in designing, coding, and maintaining C++ applications independently.

# Virtual C++ Laboratory

Interactive virtual lab for learning C++ programming concepts through theory, simulations, quizzes, and step-by-step execution of programs.

**Tagline:** Learn, Practice, and Master C++ Programming Through Interactive Virtual Experiments
## Structure

```
├── index.html          Introduction
├── objective.html      Lab objectives
├── LOE.html            List of experiments (10 OS experiments)
├── taraud.html         Target audience
├── coal.html           Course alignment
├── feed.html           Feedback
├── Exp1/ ... Exp10/    Experiment modules (same structure as original OOPS lab)
│   ├── AO.html         Aim
│   ├── theory.html     Theory
│   ├── quiz.html       Pretest
│   ├── pro.html        Procedure
│   ├── sim.html        Simulation
│   ├── postt.html      Posttest
│   ├── ref.html        References
│   └── feed2.html      Feedback
```


## How to Run

Open `index.html` in a web browser. No server or build step required.

## UI

Same Virtual Labs UI as the original OOPS project — hamburger menu, orange accent bar, modal navigation, and simulation grid layout (Start / Next / Reset).

## Simulations

Each experiment includes an interactive simulator with step-by-step execution, visualization panel, and performance metrics where applicable.
